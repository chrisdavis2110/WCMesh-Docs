# Mesh Bots

This section covers automated bots and tools that help manage and interact with the MeshCore network, making it easier to monitor repeaters, manage keys, and coordinate network resources.

## Available Bots

### MeshBuddy (Discord)

**[Learn about MeshBuddy â†’](./meshbuddy.md)**

MeshBuddy is a Discord bot that provides comprehensive repeater management and network monitoring capabilities directly from Discord channels. It's the primary tool for managing repeater prefixes, checking network status, and coordinating key assignments.

**Key Features:**
- **Repeater Management**: List all repeaters, check status, and view detailed statistics
- **Prefix Management**: Check availability, reserve prefixes, and manage key assignments
- **Network Monitoring**: Track offline repeaters, identify duplicates, and monitor network health
- **Key Generation**: Generate public/private key pairs for new repeaters
- **QR Code Generation**: Create QR codes to easily add repeaters to your contact list

**Common Use Cases:**
- Finding an available hex prefix for a new repeater
- Checking if a repeater is online or offline
- Reserving a prefix before setting up a new repeater
- Generating keys when changing repeater configurations

## Getting Started with MeshBuddy

MeshBuddy is available in the West Coast Mesh Discord server. To use it:

1. **Join Discord**: Join the [West Coast Mesh Discord](https://discord.gg/wcmesh)
2. **Navigate to the appropriate channel**: Find your region's `#repeater-control` channel
3. **Start using commands**: Type `/` followed by a command name to see available options
4. **Get help**: Use `/help` to see all available commands and their descriptions
